Experiment 4 A plain text format for composing and arranging patterns.

format

notes defines the available midi notes

notes 60 72 make a scale with middle c and the c above

clock sets the time (in seconds) for each pattern

clock 1 each pattern uses 1 second

[a-z] define a pattern

a 0 1 define a pattern (with the name 'a') that is the first and second note in the scale

| create a track

| a a . a play pattern a twice, rest, then play pattern a again

. rest: rest for the length of a note (is in a pattern) or a pattern (if in a track)

- sustain: hold the previous note in a pattern

* wildcard: play a random note (if in a pattern) or a random pattern (if in a track)

@ loop: loop a pattern multiple times
